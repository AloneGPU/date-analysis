# ESP8266 分贝检测工具

## 📋 项目介绍

本项目实现了一个基于ESP8266的分贝检测工具，支持以下两种方案：

### 方案1：INMP441 + ESP32（推荐）
- 高精度MEMS数字麦克风
- I2S接口
- 真正的数字音频处理
- 最佳精度和性能

### 方案2：MAX4466 + ESP8266（实际可用）
- 模拟麦克风模块
- 连接到模拟输入A0
- 更简单，更兼容ESP8266

### 方案3：模拟数据模式（测试用）
- 不依赖硬件
- 用于测试连接和数据显示

## 🔌 硬件连接

### 方案2：MAX4466 + ESP8266 接线图

```
MAX4466      ESP8266
OUT       →  A0 (模拟输入)
VCC       →  3.3V 或 5V
GND       →  GND
GAIN      →  可选（连接到VCC增加增益）
```

### 方案1：INMP441 + ESP32（更推荐）
由于ESP8266的I2S支持有限，建议使用ESP32：

```
INMP441      ESP32
VDD       →  3.3V
GND       →  GND
L/R       →  GND (左声道)
WS        →  GPIO25
SCK       →  GPIO26
SD        →  GPIO32
```

## 🚀 快速开始

### 第一步：准备代码

1. 打开 `esp8266_inmp441_simple/esp8266_inmp441_simple.ino`
2. 配置WiFi信息（修改 `wifi_config.h`）

### 第二步：选择工作模式

在代码中修改：
```cpp
bool useSimulatedData = true;   // 模拟数据模式（测试用）
// 或
bool useSimulatedData = false;  // 真实麦克风模式
```

### 第三步：上传代码

使用Arduino IDE上传到ESP8266

### 第四步：连接到Flutter应用

1. 打开你的数据监控应用
2. 选择"WiFi"模式
3. 扫描设备或手动输入IP地址
4. 连接并查看实时分贝数据

## 📊 数据格式

### 发送到Flutter应用的JSON格式：
```json
{
  "device_id": "esp8266_db_001",
  "timestamp": 1234567890,
  "data": {
    "decibel": 55.3,
    "avg_decibel": 54.2,
    "max_decibel": 72.5,
    "min_decibel": 35.1
  }

```

```
[分贝] 当前: 55.3 dB | 平均: 54.2 dB | 最大: 72.5 dB | 最小: 35.1 dB
```

## 🎯 分贝参考值

| 分贝值 | 声音环境 |
|-------|---------|
| 0-20 | 非常安静（静夜） |
| 20-40 | 安静（图书馆） |
| 40-60 | 正常（日常对话） |

## 🔧 校准说明

### 使用真实麦克风时需要校准

1. 在安静环境中运行
2. 记录显示的分贝值
3. 调整代码中的偏移值：

```cpp
currentDecibel += 30;  // 调整这个值进行校准
```

### 校准步骤

1. 用专业声级计测量环境噪音
2. 对比ESP8266的显示值
3. 调整偏移量直到两者一致

## 💡 使用建议

### ESP8266限制

⚠️ ESP8266对I2S支持有限：
- 有些库可能不兼容
- 可能需要额外的硬件
- 性能不如ESP32

### 更好的选择：ESP32

建议使用ESP32获得更好的性能：
- 原生I2S支持
- 更多引脚
- 更强的处理能力

### 替代方案

如果INMP441 + ESP8266有问题，可以：
1. 使用MAX4466模拟麦克风（方案2）
2. 使用ESP32替代ESP8266
3. 先使用模拟数据模式测试

## 🔍 故障排除

### 问题1：串口监视器没有数据

**解决：**
- 检查波特率是否为115200
- 确认串口选择正确
- 查看是否有"模拟数据模式"提示

### 问题2：WiFi连接失败

**解决：**
- 检查WiFi名称和密码
- 确认WiFi是2.4GHz
- 尝试靠近路由器

### 问题3：Flutter应用收不到数据

**解决：**
- 确认在同一WiFi网络
- 检查防火墙设置
- 尝试手动输入IP连接

### 问题4：分贝值异常

**解决：**
- 检查麦克风连接
- 调整校准偏移
- 确保电源稳定
