# 数据监控助手 - 功能说明与使用指南

## 📋 项目概述

数据监控助手是一款功能强大的跨平台上位机应用，支持在 Android 手机和 PC 端（Windows、macOS）运行，具备数据接收、可视化、存储等核心功能。

## 🚀 使用流程

### 1. 连接设备
1. 打开应用
2. 点击右上角设置图标
3. 选择通讯方式（MQTT/蓝牙/WiFi）
4. 输入配置信息；WiFi 方式可以扫描局域网数据设备后选择填入
5. 点击连接按钮

### 2. 接收数据
1. 确保设备已连接
2. 设备开始发送数据
3. 数据自动显示在主界面

### 3. 查看可视化
1. 点击"查看更多图表"按钮
2. 选择不同的图表类型
3. 切换数据指标

### 4. 管理数据
1. 点击存储设置图标
2. 查看数据统计
3. 导出或清理数据

## 📝 数据格式要求

应用支持 JSON 格式的数据：

```json
{
  "device_id": "sensor_001",
  "timestamp": 1640000000000,
  "data": {
    "temperature": 25.5,
    "humidity": 60.0,
    "voltage": 3.3,
    "current": 0.5
  }
}
```

## ⚠️ 注意事项

1. **网络连接**：确保设备与应用的网络连接正常
2. **蓝牙权限**：Android 设备需要授予蓝牙权限
3. **数据格式**：确保发送的数据为有效的 JSON 格式
4. **同时连接**：同一时间只能使用一种通讯方式
5. **数据存储**：定期导出重要数据以防丢失

WiFi 局域网发现协议见 [docs/wifi_discovery_protocol.md](docs/wifi_discovery_protocol.md)。

## 🔍 故障排除

### 连接失败
1. 检查网络连接
2. 确认设备地址和端口正确
3. 检查防火墙设置
4. 查看错误提示信息

### WiFi 扫描不到设备
1. 确认手机或电脑与采集设备在同一局域网
2. 确认固件已按 WiFi 发现协议回复 UDP 探测包
3. 检查路由器是否开启 AP 隔离或客户端隔离
4. Windows 端确认防火墙允许应用访问专用网络
5. 可手动输入设备 IP 和端口进行连接

### 数据不显示
1. 确认数据格式正确
2. 检查 Topic 名称
3. 查看设备是否正在发送数据

### 蓝牙扫描不到设备
1. 确保设备已配对
2. 检查手机蓝牙是否开启
3. 检查位置权限是否授予
